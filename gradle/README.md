# Gradle Configuration

Use the Gradle Wrapper (`gradlew`) to ensure consistent build environments.

## Common Tasks

*   `./gradlew assembleDebug`: Build Android App.
*   `./gradlew run`: Run Desktop App.
*   `./gradlew packForXcode`: Build iOS Framework (called automatically by Xcode).

## Versions

Dependencies and versions are managed in `gradle/libs.versions.toml`.
